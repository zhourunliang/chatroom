### 聊天室
+ Flask-SocketIO的聊天室
### 本地环境配置
+ 安装python3
+ 安装flask和flask_socketio相关的库
+ 克隆项目到本地
```bash
git clone https://github.com/zhourunliang/chatroom
```
+ 运行程序
```bash
python chat.py
```
+ 用浏览器访问http://127.0.0.1:5000
